class hercules_sync.git.GitDataLoader(repo_name, before_ref, after_ref)[source]¶

Bases: object

Downloads github files from a commit, before and after a push.

This class is used to download files from a github repository given a certain original (before_ref) and final commit (after_ref).

Parameters

  • repo_name (str) – Full name of the github repository where the files are located.

  • before_ref (str) – Sha of the initial commit before the push.

  • after_ref (str) – Sha of the final commit after the push.
