Escenarios¶

A continuación se describen los escenarios probados, utilizando el framework Cucumber.

Feature Descripción
change_triplestore.feature Caso de uso donde se comprueba que el sistema funciona con distintos triplestores.
synchronize_rdf.feature Caso de uso donde se comprueba que el sistema es capaz de propagar los cambios que se realicen en la ontología al triple store elegido. En este caso wikibase.
